@h4 Drop a Table
@link https://www.postgresqltutorial.com/postgresql-tutorial/postgresql-drop-table/
@start-textarea
DROP TABLE [IF EXISTS] table_name 
[CASCADE | RESTRICT];
@end-textarea

@start-pre
1 - The CASCADE option allows you to remove the table and its dependent objects.
2 - The RESTRICT option rejects the removal if there is any object depends on the table. The RESTRICT option is the default if you donâ€™t explicitly specify it in the DROP TABLE statement.
