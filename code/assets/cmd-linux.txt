# Resource Utilization

## list all processes
### ps -ef

## display sorted information about processes
### top
### top -o +%MEM
### top -o +%CPU

## virtual memory
### vmstat

## list of open files
### lsof

# Networking

## netstat - dentify open ports, connections to external IP addresses, and the connection status
### netstat


## tcpdump - check network traffic
### sudo tcpdump
### sudo tcpdump port 443

# Disk usage

## list disk usage for a path (human readable)
### du -sh

## list disk volume utilization (human readable)
### df -h
