@h3 Singly Linked List
@img https://upload.wikimedia.org/wikipedia/commons/thumb/6/6d/Singly-linked-list.svg/408px-Singly-linked-list.svg.png
@h4 Operations
@startpre
1 - insert after a given node or value, insert at start, insert at end
2 - delete a given node or value, delete at start, delete at end
3 - traverse natural order
4 - find a node or value in the linked list
5 - find length of the linked list
@endpre

---

@h3 Doubly Linked List
@img https://upload.wikimedia.org/wikipedia/commons/thumb/5/5e/Doubly-linked-list.svg/610px-Doubly-linked-list.svg.png
@h4 Operations
@startpre
1 - insert after a given node or value, insert at start, insert at end
2 - delete a given node or value, delete at start, delete at end
3 - traverse natural or reverse order
4 - find a node or value in the linked list
5 - find length of the linked list
@endpre
